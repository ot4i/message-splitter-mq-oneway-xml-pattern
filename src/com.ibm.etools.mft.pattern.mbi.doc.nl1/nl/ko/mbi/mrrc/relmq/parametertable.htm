입력 메시지의 유효성 검증 아니오 예 없음 이 패턴 매개변수는 입력 메시지에 필요한 유효성 검증 레벨을 식별합니다. 올바른 값은 다음과 같습니다.
  • 없음
  • 컨텐츠
  • 컨텐츠 및 값
유효성 검증이 없음으로 설정되면 메시지가 불명료한 BLOB로 처리됩니다.

유효성 검증이 컨텐츠 또는 컨텐츠 및 값으로 지정되면, MQInput 노드를 구성하는 데 사용되는 추가 입력 정보가 필요합니다.

BAR 파일에서 이 특성을 구성할 수 있습니다. 이 특성을 구성하면 유효성 검증을 해제할 수 있습니다.

요청 메시지 매개변수가 정의되어 있지 않으면 이 특성을 사용하여 유효성 검증을 설정할 수 없습니다.
입력 데이터 유형 아니오 아니오 XML 이 패턴 매개변수는 입력 메시지의 유효성 검증 값이 컨텐츠 또는 컨텐츠 및 값인 경우에만 필요합니다. 올바른 값은 다음과 같습니다.
  • XML
  • 2진 데이터(예: C 또는 COBOL 구조)
  • 텍스트 데이터(예: CSV, SWIFT 또는 HL7)

최대 응답 시간(1/10초 단위) 예 예 제한시간 없음 이 패턴 매개변수는 응답을 대기하는 최대 시간을 정의합니다.

기본값은 제한시간 없음입니다. 그러나 동일한 필드에 제한시간 값으로 양의 정수를 입력할 수 있습니다. 제한시간은 1/10 초로 측정되며 저장된 MQMD 헤더를 보관한 메시지는 이 만기 시간까지 저장됩니다.

값이 제한시간 없음이면 브로커로 리턴되는 응답이 항상 클라이언트로 리턴됩니다.

이 패턴 매개변수는 큐에 남겨진 메시지의 자동 보관을 제공하지만 해당 기간이 경과한 후 응답을 클라이언트와 일치시키지 못하도록 합니다.

저장 큐 예 예 STORE 이 패턴 매개변수는 제공자 응답을 대기하는 동안 MQMD 헤더를 저장하는 데 사용되는 큐를 정의합니다.

이 큐는 브로커에 대해 로컬입니다.
