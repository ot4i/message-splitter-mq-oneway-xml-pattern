Modèle d'Utilitaire de fractionnement de message pour WebSphere MQ : unidirectionnel (pour XML)

Utilisez le modèle Utilitaire de fractionnement de message pour WebSphere MQ : unidirectionnel (XML) pour diviser un message XML volumineux en plus petits éléments traités par une ou plusieurs cibles en utilisant des flux transactionnels et des messages WebSphere MQ persistants.

Utilisez ce modèle pour les applications qui stockent des informations sur des transactions métier et transmettent ces informations à une ou plusieurs applications cibles dans des lots. Vous pouvez l'utiliser pour gérer les messages volumineux sans utiliser excessivement la mémoire.

Vous pouvez utiliser ce modèle lorsque l'application source ou les applications cible ne peuvent pas créer ou traiter des appels de service quasi temps réel ou lorsque la répartition en lots des informations s'impose pour des raisons professionnelles.

Solution

La solution consiste à implémenter un flux de messages qui utilise une arborescence de messages modifiable permettant de supprimer les éléments reproductibles au fur et à mesure qu'ils sont traités. L'analyse partielle permet d'analyser des éléments individuels en fonction des besoins et, du fait que ces éléments sont supprimés à la fin, l'arborescence dans la mémoire interne ne croît pas de manière linéaire avec le nombre d'éléments. Chaque élément extrait est routé vers l'application cible appropriée.
