Flux secondaire Journal pour le modèle d'utilitaire de Fractionnement de message pour WebSphere MQ : unidirectionnel (pour XML)

Les messages de journal sont écrits sous forme de messages persistants WebSphere MQ dans une file d'attente WebSphere MQ.

Ces messages de journal peuvent être traités séparément en fonction des pratiques standard de votre entreprise. Le journal de messages contient le message d'origine et les informations suivantes dans un en-tête MQRFH2 :

  • Horodatage
  • Nom du courtier
  • Nom du flux de messages
  • Etat (0 succès)
